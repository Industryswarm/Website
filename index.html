ISNode

ISNode is a solid and innovative framework for building scalable enterprise products. It can be used to build monolithic applications or microservice based architectures and contains many innovations - (i) a multi-tenant Identity component with full OAuth2 and SSO capabilities; (ii) extendable interfaces for your services including HTTP/S, WebSockets, ZeroMQ, NanoMSG, SSH and more; and (iii) An extendable ORM data engine for connecting to almost any data repository.
